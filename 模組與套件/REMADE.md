    可透過import引用其他檔案，就可以使用其他檔案內的類別與函數進行功能擴充。 
    模組就是一個python檔案，模組名稱就是一個檔案名稱 
    Python檔案命名時不要與其他已知模組名稱相同，名稱相同時預設呼叫自己的檔案。  
    
# import模組語法

  1.import 模組   
  2.import 模組 as 新模組名稱  
  3.from 模組 import 模組內方法  
  4.不建議 from 模組 import *，易造成名稱衝突
